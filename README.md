# 🏆 Universal Tournament Manager

A full-stack tournament management application built with **Python Flask** — no Node.js required.  
Works for **any sport**: cricket, football, basketball, tennis, and more.

---

## 🚀 Features

- **Public Pages**: Home, Live Matches, Standings, Teams, Bracket  
- **Admin Panel**: Create tournaments, manage teams/players, generate schedules, live score entry  
- **Real-time**: Socket.IO for live score updates  
- **Dark Theme**: Neon-glow glassmorphism UI with 15+ CSS animations  
- **JWT Auth**: Secure admin access with httpOnly cookie tokens  
- **SQLite**: Zero-config database (built into Python)

---

## 🧱 Tech Stack

| Layer     | Technology |
|----------|-----------|
| Backend  | Python 3.10+ / Flask / Flask-SocketIO |
| Database | SQLite (built-in `sqlite3`) |
| Auth     | PyJWT + werkzeug password hashing |
| Frontend | Jinja2 templates + Tailwind CSS CDN |
| Real-time| Socket.IO (Flask-SocketIO + CDN client) |
| Animations | Pure CSS @keyframes + Intersection Observer |
